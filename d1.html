Jewelry

This project involved analyzing and organizing inventory data in Google Sheets for a fictional jewelry business. I imported a CSV dataset and built a structured dashboard using formatted titles, organized categories, and calculated metrics. Using functions such as COUNTIF, SUMIF, AVERAGEIF, MIN, MAX, and RANK, I generated insights including unique items in stock, total quantities, inventory value, average pricing, and price rankings. I also created an “Inventory Value” column that automatically calculated the dollar value of each item based on quantity and retail price. This project strengthened my skills in data cleaning, spreadsheet formulas, and building functional analytical sheets that summarize business inventory data clearly and efficiently.
